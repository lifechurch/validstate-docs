# Validstate Documentation Site
This is the code base for the documentation of the [Validstate](https://github.com/lifechurch/validstate) package. 

Please note: This repo does not contain the source code for Validstate only the documentation site.

## Contributing
If you would like to contribute to the documentation because you have make a change to the Validstate repo or would like to fix an issue with the site here are the steps. 

### Prerequisits
This is a Ruby on Rails application if you are not familiar with Rails projects checkout [Getting Started with Rails](http://guides.rubyonrails.org/getting_started.html).

Please ensure you have the correct version of Ruby installed, as well as the bundler, and foremen gems. 

### Running the application locally
1. Fork the repository and create your branch from master.
1. Clone your fork to your machine
1. Run `bundle install` 
1. Run `npm install`
1. Run `foreman start -f Procfile.dev`
